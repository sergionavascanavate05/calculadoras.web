Tokens de border-radius y comportamiento de sombras del proyecto fuente.

Border Radius

8px

--radius-lg
12px
Tarjetas, formularios, secciones
--radius-full
9999px
Badges, círculo IMC, toggle

Elevación / Sombras

El proyecto fuente usa sombras mínimas. Solo se aplica shadow-sm en hover de CalculatorCard. No hay sombras fijas en estado normal.

Sin sombra (default)
Estado normal de tarjetas y formularios. Solo borde.
shadow-sm (hover)
Hover de CalculatorCard. border-color + shadow.
focus ring
Input focus: 2px accent al 30%.

Contexto de uso en el proyecto

rounded-xlCalculatorCard, formularios, secciones de blog, resultado (12px)
rounded-lgBotones, enlaces de navegación, theme toggle (8px)
rounded-fullCírculo de resultado IMC (80px, 9999px)
hover:shadow-smCalculatorCard hover (border + shadow)
focus:ring-2Input focus: ring accent/30
